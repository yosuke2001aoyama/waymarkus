# App Store Screenshot Set

Capture these from the signed release build on an iPhone 17 Pro Max, iPhone 16 Pro Max, or another device/simulator that produces an accepted 6.9-inch size.

Preferred portrait canvas: `1320 × 2868` pixels.

Apple accepts one to ten screenshots. Use five:

1. **Understand what you see** — Home with the primary Ask action.
2. **Ask about the place in front of you** — A sourced Taos or Louisville answer, including “What not to assume.”
3. **Say the thought before it disappears** — Native Capture with the microphone button and private status.
4. **Map what you noticed** — Native Memory Map with two or more clearly fictional test notes.
5. **Your private field library** — Native Library with search and several fictional records.

## Capture Rules

- Use fictional notes and no private names, exact itineraries, or personal locations.
- Keep the status bar clean and use a stable Wi-Fi connection.
- Use the production server and release configuration.
- Do not show permission alerts, debugging overlays, Safari chrome, or developer tools.
- Screenshots must show the real app UI. Promotional text can be added outside the device UI only after the raw captures are preserved.
- Store final PNG files in this directory using `01-home.png` through `05-library.png`.
